# Argus Lab Documentation Index

---

## Overview

This directory contains the planning, architecture, training, validation, and operations documentation for Argus Lab.

Argus Lab is a planned real Linux troubleshooting, training, and validation environment.

It has two connected purposes:

- help people build real Linux troubleshooting skill through realistic failure scenarios
- validate NeuroCore, Argus ACLI, and model-guided troubleshooting against known Linux faults
